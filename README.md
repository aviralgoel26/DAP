# DAP - Document Automation Platform

## Overview

DAP is an enterprise document automation platform that generates customized Word and Excel documents from predefined templates.

Version: 1.0

Current Development Phase:
Phase 1 - Template Automation

## Tech Stack

Backend:
- Java 21
- Spring Boot
- Maven

Frontend:
- React

Libraries:
- Apache POI

## Project Structure

backend/
frontend/
templates/
generated/
uploads/
docs/

## Current Feature

Generate customized documents by replacing:

- Company Name
- Company Address
- Company Logo
